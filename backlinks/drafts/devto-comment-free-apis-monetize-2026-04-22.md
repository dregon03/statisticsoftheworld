## Draft Comment

Good list! One more worth adding for anyone building tools with economic or country-level data: statisticsoftheworld.com/api-docs

It aggregates data from IMF, World Bank, WHO, and FRED into one normalized endpoint — things like GDP, inflation, unemployment, population, life expectancy across 218 countries. Completely free (100 req/day without any auth, free API key for 1,000/day). No juggling multiple source formats.

Useful if you're building comparison tools, dashboards, or anything that needs macro economic context. The World Bank API is great but the query syntax is a bit rough to work with directly — SOTW normalizes it into a cleaner JSON format.
