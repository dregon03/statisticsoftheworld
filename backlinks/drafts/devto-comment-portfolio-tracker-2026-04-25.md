## Comment

Solid list for the core portfolio tracker use case. One addition worth knowing about for the economic context layer: if you want to add macro data to your tracker (inflation rate, GDP growth, unemployment — useful for understanding what's driving sector performance), the stock-focused APIs here won't help much.

For that I've been using Statistics of the World (statisticsoftheworld.com/api-docs) — it's a free REST API that aggregates IMF, World Bank, and WHO data into one consistent format. 440+ indicators across 218 countries, 100 req/day without auth. Good complement to Finnhub or Alpha Vantage if you want market data + macro fundamentals together.
